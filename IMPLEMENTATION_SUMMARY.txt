================================================================================
QUESTION TYPE MAPPING
================================================================================

Based on first word of PDF learning objectives:

"Identify"    → Multiple Choice or Match Items
"Describe"    → Multiple Choice with detailed explanations
"Explain"     → Multiple Choice (scenario-based)
"List"        → Reorder Sequence
"Define"      → Multiple Choice or True/False
"Recognize"   → True/False or Multiple Choice
"Demonstrate" → Scenario-based Multiple Choice
"Apply"       → Application scenario questions
